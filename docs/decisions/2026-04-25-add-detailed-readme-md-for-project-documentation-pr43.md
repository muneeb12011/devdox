### Architecture Decision Record

**Problem:**
Project documentation was lacking, causing difficulties for new contributors and users.

**Decision:**
Improve project documentation by adding a detailed README.md.

**Context:**
This change aims to assist new contributors and users in understanding the project.

**Consequences:**
Legacy auth flow might break due to this change.

**Rationale:**
We chose to improve project documentation to address the lack of information for new contributors and users.

**Status:**
Accepted
