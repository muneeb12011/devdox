# ADR

## Problem
The current auto-formatting tools used in README.md were causing formatting issues, making it difficult to read and understand the content.

## Context
The README.md file is a critical component of the project, providing essential information to users and contributors.

## Decision
We decided to use manual formatting in README.md to ensure consistency and readability.

## Consequences
This decision may lead to a higher maintenance burden for the README.md file, but it will provide a more consistent and readable experience for users and contributors.

## Status
Proposed
